# Eliyannah Amirah Yisrael — Writer / Director

The personal site of Eliyannah Amirah Yisrael. A Sunshine Moxie Production under The Forest of God's Answers.

Live site: [sunshinemoxie.co](https://sunshinemoxie.co)

## Stack

- **Framework:** [Astro](https://astro.build) — static site generator
- **CMS:** Keystatic (added in next phase)
- **Hosting:** Netlify
- **Newsletter:** Buttondown (handled separately)

## Local Development

```bash
# Install dependencies
npm install

# Start dev server (runs at localhost:4321)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
  pages/        # Each .astro file = one page
  layouts/      # Reusable page wrapper
  components/   # Header, footer, nav, etc.
  styles/       # Global CSS
public/
  assets/       # Images, fonts, static files
```
